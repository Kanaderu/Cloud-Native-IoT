##########
Kubernetes
##########

Kubernetes is an open-source platform for container orchestration developed by Google and now a project of the Cloud Native Computing Foundation (CNCF). Kubernetes is a system that encompasses many concepts which can easily take years to master the system as a whole. However, consistent key concepts are reviewed to realize the benefit and ease of use of Kubernetes. As a gentle introduction to Kubernetes, here is a Kubernetes story to help introduce some concepts.
